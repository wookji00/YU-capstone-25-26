### 라즈베리파이 내부
#### bash로 블루투스 관리자 들어가기
```bash
sudo bluetoothctl # 블루투스 관리자
>> scan on
>> pair MAC주소 # 옆에 HC-06이라고 뜸. 이름 왼쪽이 바로 MAC addr
>> trust MAC주소
>> exit quit 	# bluetoothctl 관리자를 통해 HC-06을 계속 쓰기 위함
```
#### agent off 이후 bash
```bash
sudo rfcomm connect 0 MAC주소
sudo rfcomm bind 0 MAC주소 # rfcomm으로 블루투스 연결 시도
ls /dev/rfcomm* 	# (보통 붙는 이름) rfcomm0 이 뜨면 연결 성공!
```

## 3. P-tank 연결

  - https://github.com/koadrone/ardupilot/releases/tag/KoaFC-Ardupilot 링크에서 .apj 파일 다운로드받기 (ArduRover.apj)

### 3-1. Mission Planner 로 펌웨어 update

  - Mission Planner-를 설치 및 실행합니다.
  - FC 에 USB 를 연결합니다.
  - 미션 플래너 화면 우측 상단에서 알맞은 COM 번호(PC 마다 다름)를 선택하고, 통신 속도는 115200 을 선택합니다. 연결 버튼은 누르지 않습니다.
  - 펌웨어 설치 페이지에서 Load custom firmware 를 클릭하여 펌웨어를 선택합니다.
  Upload Done 메세지가 뜨면 펌웨어 설치가 완료된 것입니다. FC 에서 USB 를 분리합니다.

#### **Mission Planner on Linux**

  It is possible to run most Windows based programs on many Linux Distributions using MONO. Mission Planner does run under MONO but will have occasional issues and/or crashes. QGC and MAVProxy are alternatives that run stably in Linux, but if Mission Planner is really desired, you can follow these steps:

  Download and install the latest version of MONO from [다음 명령어]
  ```bash
  sudo apt install ca-certificates gnupg
  sudo gpg --homedir /tmp --no-default-keyring --keyring gnupg-ring:/usr/share/keyrings/mono-official-archive-keyring.gpg --keyserver hkp://keyserver.ubuntu.com:80 --recv-keys 3FA7E0328081BFF6A14DA29AA6A19B38D3D831EF
  sudo chmod +r /usr/share/keyrings/mono-official-archive-keyring.gpg
  echo "deb [signed-by=/usr/share/keyrings/mono-official-archive-keyring.gpg] https://download.mono-project.com/repo/ubuntu stable-focal main" | sudo tee /etc/apt/sources.list.d/mono-official-stable.list
  ```
  ```bash
  sudo apt update
  # Download Mission Planner as a zip file from [here|https://koafc.com/supported-firmware/ardupilot] and unzip to a directory.
  ```
#### 디렉토리 바꾸기

### 3-2. 라즈베리와 KOA FC 데이터 통신을 위한 MAVproxy 설치법
```bash
sudo apt update
sudo apt install -y python3-pip python3-dev python3-opencv python3-wxgtk4.0 libxml2-dev libxslt1-dev
```
# MAVproxy pip로 설치: python 가상환경 필요
```bash
pip3 install pyserial
pip3 install mavproxy
```
```bash
# USB로 연결
mavproxy.py --master=/dev/ttyACM0 --baudrate=115200 --out=udp:10.42.0.255:14550
```
또는
```bash
# UART 연결
mavproxy.py --master=/dev/ttyAMA0 --baudrate=57600 --out=udp:10.42.0.255:14550
```
#### "MAV> " 가 뜨고 데이터가 흐르기 시작하면 성공!
